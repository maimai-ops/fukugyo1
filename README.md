# LP Sample (Next.js App Router)

## 概要
Next.js（App Router） + TypeScript + Tailwind CSS で作成した、1ページ構成のランディングページ（LP）です。  
スマホ優先のレスポンシブ対応で、API連携なし（ダミーデータ）で要件を満たしています。

## ページ構成
- ファーストビュー（キャッチコピー＋簡単な説明）
- サービスの特徴（3点）
- 利用シーン（文章）
- お問い合わせ / 近日公開 表記

## 技術スタック
- Framework: Next.js (App Router)
- Language: TypeScript
- CSS: Tailwind CSS
- Data: ダミーデータ（API連携なし）

## 起動方法
```bash
npm install
npm run dev
